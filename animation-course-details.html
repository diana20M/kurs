Лекция 1: Основы анимации

1.1 Что такое компьютерная анимация?

Компьютерная анимация - это процесс создания движущихся изображений с использованием компьютерных технологий. Она широко используется в кино, играх, рекламе и мультимедиа.

2D анимация (плоское изображение)

3D анимация (объемное изображение)

1.2 12 принципов анимации

Основополагающие принципы, разработанные аниматорами Disney:

Сжатие и растяжение (Squash and Stretch)

Придает объектам ощущение веса и гибкости при движении.

Подготовка (Anticipation)

Движение перед основным действием, подготавливающее зрителя.

Совет: Эти принципы применимы как к 2D, так и к 3D анимации.

3.2 Ключевые кадры и интерполяция

Основные понятия в 3D анимации:

Ключевые кадры (Keyframes)

Кадры, где заданы основные положения объекта. Программа автоматически рассчитывает промежуточные кадры.

Кривые интерполяции

Графики, определяющие как изменяются параметры между ключевыми кадрами (линейно, с ускорением и т.д.).

Пример кода Maya MEL:

// Создание ключевого кадра для перемещения
setKeyframe -breakdown 0 -hierarchy none -controlPoints 0 -shape 0 {"pCube1.translateX"};
// Настройка интерполяции
keyTangent -inTangentType linear -outTangentType linear;
