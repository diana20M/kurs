Лекция 1: Введение в Blender

1.1 Что такое Blender?

Blender - это бесплатный и открытый 3D-редактор, который поддерживает весь 3D-конвейер: моделирование, риггинг, анимацию, симуляцию, рендеринг, композитинг и отслеживание движения.

1.2 Основные элементы интерфейса

Интерфейс Blender состоит из нескольких основных областей:

  • 3D Viewport - основная рабочая область
  • Outliner - список всех объектов в сцене
  • Properties - панель свойств объекта
  • Timeline - для работы с анимацией

Совет: Используйте колесико мыши для масштабирования и среднюю кнопку мыши для вращения вида.

Лекция 2: Основы моделирования

2.1 Работа с примитивами

Blender предоставляет несколько базовых объектов (примитивов):

Куб - основа для многих моделей

Сфера - для круглых объектов

Цилиндр - для трубчатых форм

2.2 Режимы редактирования

В Blender есть два основных режима работы с объектами:

  1. Object Mode - для работы с целыми объектами
  2. Edit Mode - для редактирования вершин, ребер и граней

Практическое задание

Создайте композицию из трех разных примитивов. Попробуйте перемещать, вращать и масштабировать их.

Лекция 3: Модификаторы

3.1 Что такое модификаторы?

Модификаторы - это автоматические операции, которые неразрушающим способом влияют на объект.

3.2 Популярные модификаторы

Модификатор Описание
Subdivision Surface Добавляет детализацию, сглаживая поверхность
Mirror Создает зеркальную копию объекта
Array Создает массив копий объекта
Boolean Объединяет или вычитает объекты

Пример кода:

# Пример использования модификатора в Python API
import bpy
obj = bpy.context.object
mod = obj.modifiers.new(name="MySubdiv", type='SUBSURF')
mod.levels = 2
